# 阶段 6：账号设置与个人资料

## 目标

阶段 6 在原有“管理员用户名与密码修改”的基础上，扩展为完整的账号设置能力，并让公开资料直接驱动前台个人博客首页。

## 后台设置

- 后台新增 `账号设置` 导航入口。
- 设置页拆分为两个子标签：`安全` 与 `资料`。
- `安全` 负责后台登录用户名、当前密码校验、新密码与确认密码。
- `资料` 负责前台公开资料，包括账号名称、头像、身份属性、个性签名、身份/职业、个人简介、技术栈、GitHub 地址、所在地。

## 前台联动

- 首页从单纯知识库集合升级为个人博客首页。
- 首屏全屏展示账号名称、头像、签名、身份属性、职业与技术栈。
- 向下滚动进入文章搜索与知识库集合区域。
- 右侧栏展示更完整的个人资料。
- 资料修改后应同步影响前台公开首页。

## 测试要求

- Playwright 覆盖后台安全设置保存。
- Playwright 覆盖后台资料修改后前台首页同步展示。
- Playwright 保留前台知识库、搜索和文章阅读主流程。
