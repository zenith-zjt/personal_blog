# 前端基础知识库总览

欢迎进入前端基础知识库。
这里的内容采用 Markdown 文件组织，知识库树直接来源于文件系统目录结构。

![知识地图](knowledge-map.svg)

## 当前约定

| 项目 | 说明 |
| --- | --- |
| 一级目录 | 视为一个知识库 |
| 子目录 | 视为知识库内部层级 |
| Markdown | 文章唯一可信源 |
| resource | 同级资源目录，前台树中默认隐藏 |

- 一级目录代表知识库
- 子目录代表知识树结构
- Markdown 文件代表文章
- 同级 `resource` 目录用于存放图片等文章资源
